# 📊 Live API Data Integration with Power BI

## 📌 Project Overview
This project demonstrates integration of live API data using Python and visualizing it in Power BI dashboards.

## 🎯 Objectives
- Fetch real-time data using API
- Process data using Python
- Load cleaned data into Power BI
- Create interactive dashboards

## 🛠 Tech Stack
- Python
- APIs
- Power BI
- Pandas

## 📈 Dashboard Highlights
- Economic Indicators
- Environmental Metrics
- Health & Technology Data
- Trade & Poverty Analysis

## 🚀 How to Run
1. Execute Python script to fetch API data
2. Load generated CSV into Power BI
3. Open .pbix file to view dashboard
