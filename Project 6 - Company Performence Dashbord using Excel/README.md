# 📊 Company Performance Dashboard using Excel

## 📌 Project Overview
Developed an interactive Excel dashboard to analyze company performance including sales, customers, and product data.

## 🎯 Objectives
- Build KPI-based dashboard
- Analyze customer behavior
- Track product sales
- Use pivot tables & charts

## 🛠 Tech Stack
- Microsoft Excel
- Pivot Tables
- Charts
- Data Modeling

## 📈 Dashboard Features
- Sales Overview
- Customer Segmentation
- Product Performance
- KPI Metrics

## 🚀 How to Use
1. Open Excel dashboard file
2. Use filters to explore data
3. Analyze KPIs dynamically
