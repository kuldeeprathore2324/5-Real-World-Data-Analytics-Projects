# 🏥 Hospital Database Design & Data Migration

## 📌 Project Overview
This project involves designing a hospital database schema and performing data migration and analysis using SQL.

## 🎯 Objectives
- Create relational database schema
- Normalize hospital data
- Perform data migration
- Optimize SQL queries

## 🛠 Tech Stack
- SQL (MySQL / SQL Server)
- CSV Data
- Database Design

## 📌 Key Components
- Doctor Credentials Table
- Patient Records
- Data Cleaning Queries
- Performance Optimization

## 🚀 How to Run
1. Import SQL script into database
2. Load CSV files
3. Execute queries for analysis
